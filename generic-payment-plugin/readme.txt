Generic payment gateway integration for WooCommerce.

== Description ==
This plugin adds a generic payment gateway to WooCommerce.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/generic-payment-gateway` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the payment gateway in WooCommerce settings
